41. Command to list files in HDFS
hdfs dfs -ls /path

42. Command to create a directory in HDFS
hdfs dfs -mkdir /user/data

43. Command to upload a file to HDFS
hdfs dfs -put localfile.csv /user/data/

44. Command to download a file from HDFS
hdfs dfs -get /user/data/file.csv .

45. Command to view file content in HDFS
hdfs dfs -cat /user/data/file.csv

46. Command to remove a file from HDFS
hdfs dfs -rm /user/data/file.csv

47. Command to check disk usage
hdfs dfs -du -h /path

48. Command to check HDFS health
hdfs fsck /
