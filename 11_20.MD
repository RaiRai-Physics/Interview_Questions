11. What happens when a file is uploaded to HDFS?

When a file is uploaded:

The client contacts the NameNode.
NameNode checks permissions and decides block placement.
File is split into blocks.
Blocks are written to DataNodes.
Replicas are created based on replication factor.

12. What is rack awareness?

Rack awareness means Hadoop understands the physical rack location of DataNodes.

It places replicas across different racks so that data is safe even if an entire rack fails.

13. Why is HDFS not good for small files?

HDFS is designed for large files. Each file creates metadata in the NameNode memory.

Too many small files can overload the NameNode, causing performance problems.

14. Can we change the replication factor?

Yes.

Example command:

hdfs dfs -setrep -w 2 /path/to/file

This sets the replication factor to 2.

15. What is heartbeat in Hadoop?

A heartbeat is a signal sent by each DataNode to the NameNode to show that it is alive.

If the NameNode stops receiving heartbeats from a DataNode, it marks that DataNode as dead.

16. What is MapReduce?

MapReduce is a programming model used to process large datasets in parallel.

It has two main phases:

Map phase
Reduce phase

17. What happens in the Mapper?

The Mapper processes input data and generates intermediate key-value pairs.

Example:

Input: "apple mango apple"
Output: (apple, 1), (mango, 1), (apple, 1)

18. What happens in the Reducer?

The Reducer aggregates values for the same key.

Example:

Input: (apple, [1,1])
Output: (apple, 2)
19. What is the Shuffle and Sort phase?

Shuffle and Sort happens between Map and Reduce.

It groups mapper output by key and sends the grouped data to reducers.

Example:

Mapper output:
(apple, 1), (mango, 1), (apple, 1)

After shuffle and sort:
(apple, [1,1]), (mango, [1])

20. What is a Combiner?

A Combiner is a mini-reducer that runs after the Mapper.

It reduces data locally before sending it to the Reducer, which improves performance.

Example: In word count, the Combiner can locally sum word counts before network transfer.
