21. What is YARN?

YARN stands for Yet Another Resource Negotiator.

It manages cluster resources and schedules jobs.

22. What are the main components of YARN?

Main components:

ResourceManager
NodeManager
ApplicationMaster
Container

23. What is ResourceManager?

ResourceManager is the master service in YARN. It manages cluster resources and schedules applications.

24. What is NodeManager?

NodeManager runs on worker nodes. It manages resources on that node and reports to the ResourceManager.

25. What is ApplicationMaster?

ApplicationMaster manages the execution of a specific application or job.

For example, every MapReduce job has its own ApplicationMaster.

26. What is a container in YARN?

A container is a package of resources assigned to an application.

It includes:

CPU
Memory
Disk/network resources

27. What is Hive?

Hive is a data warehouse tool built on Hadoop. It allows users to write SQL-like queries using HiveQL.

It is mainly used for batch analytics.

28. What is Pig?

Pig is a scripting platform used for data processing in Hadoop.

Its language is called Pig Latin.

29. What is Sqoop?

Sqoop is used to transfer data between Hadoop and relational databases like MySQL, Oracle, or SQL Server.

Example:

sqoop import --connect jdbc:mysql://localhost/db --table customers --target-dir /user/data/customers

30. What is Flume?

Flume is used to collect and move large amounts of log data into Hadoop.

Example use case: Moving server logs into HDFS.
